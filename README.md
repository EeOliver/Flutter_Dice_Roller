# Dice Rolling App With Modifiers!
- The Ultimate app fpr your gambling needs!

## Features
- Multiple Dice Types
- supports RPG dices such as: d4, d6, d8, d10, d12 ad d20

### - Changeable Dice Count
- roll between 1 and 10 die at once

### - Modifiers
- add or subtract a modifier (-10 to +10) from a total roll
- how it baseially works is that if the modifier is e.g. 3. Then When you roll a dice it's lowest possible number is 4.

### Optional Negative Input
- toggle button for this

### Output
- displays a screen where you can see the output of each die.

### Roll History
- keeps a scrollable history of all previous rolls with all details

### Modern UI
- Cool gradient background with cards, sliders and even the buttons are pretty!
